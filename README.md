# Arte Óptical - Construindo Ilusões

Oficina contruindo para o evento Multivervo 2.0


## Setup

- You need to have [node](https://nodejs.org/en/) and [yarn](https://yarnpkg.com/) installed.
- `yarn install`


## Development

- `yarn dev`
- http://localhost:8080/


## Deploy

- `yarn build`
- copy folder `public` to server
